# BeLLM v1 Training Process / Plan

## Step 1. Dataset 
Currently, using en c4 dataset for initial testing of the tool

## Step 2. Tokenisation
The first step is tokenization. I took various subsets of the english c4 dataset and tokenised up to 20k tokens then pruned down to 12k.

The tokenization is quite a slow task as all text needs to be tokenised multiple times and have frequencies computed. It takes signifigant time.

For future BeLLM versions we may want more disciplines for coding and also ways to handle multilingual datasets. As a result, we'll want more tokens.

## Step 3. Foundation Model Training
The next step is training the foundational model. This is just making it an overpowered text prediction algorithm.

Train only two sizes of the model.

Start with basic small attention mechanisms. can try other ones in future.

## Step 4. Chat-based Fine Turning
Fine tune the model to get used to answering text in a chat based Q&A style approach

## Step 5. RL Learning
Train the model to use reinforcement learning by giving it a series of questions and reward it based on it's answer.
Eventually this should be the main way of training the model. That way we move more towards Reinforcment learning and less towards text generation
so that it trains itself autonomously more.
